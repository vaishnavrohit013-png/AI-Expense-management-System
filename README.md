# AI Expense Management System

A full-stack MERN application to track expenses and manage finances.

## Features
- User authentication
- OTP login
- Expense tracking
- Budget management
- Financial reports
- AI expense insights

## Tech Stack
Frontend: React + Vite  
Backend: Node.js + Express  
Database: MongoDB

## Installation

Frontend
npm install
npm run dev

Backend
npm install
npm run dev
